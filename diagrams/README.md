# PlantUML Диаграммы - MSFS AutoLand

**Дата создания:** 2026-04-16  
**Формат:** PlantUML (профессиональные UML диаграммы)

---

## 📊 Созданные диаграммы:

### 1. **classes.puml** - Диаграмма классов
Показывает все основные классы проекта, их атрибуты, методы и связи.

**Включает:**
- 18+ классов
- Все основные методы
- Связи между классами (композиция, использование)
- Enum ApproachPhase
- Аннотации и комментарии

### 2. **sequence.puml** - Диаграмма последовательности
Показывает порядок вызовов методов при выполнении захода на посадку.

**Включает:**
- Полный цикл от подключения до завершения
- Все фазы захода (INITIAL → INTERMEDIATE → FINAL → LANDING)
- Взаимодействие между компонентами
- Условные переходы (alt/else)
- Циклы (loop)

### 3. **architecture.puml** - Архитектура компонентов
Показывает компонентную архитектуру системы.

**Включает:**
- Все модули проекта
- Пакеты (packages)
- Связи между компонентами
- Цветовое кодирование
- Аннотации с описаниями

## 🚀 Как визуализировать:

### Способ 1: Online (БЫСТРО - рекомендую)

1. **Запустите скрипт:**
   ```
   visualize_plantuml.bat
   ```
   Выберите вариант `1`

2. **Или вручную:**
   - Откройте http://www.plantuml.com/plantuml/uml/
   - Скопируйте содержимое .puml файла
   - Вставьте в редактор
   - Диаграмма отобразится автоматически
   - Экспортируйте в PNG/SVG

### Способ 2: VS Code (УДОБНО)

1. **Установите расширение:**
   - Откройте VS Code
   - Extensions → Поиск "PlantUML"
   - Установите "PlantUML" by jebbs

2. **Откройте диаграмму:**
   - Откройте любой .puml файл
   - Нажмите `Alt+D` для превью
   - Или правый клик → "Preview Current Diagram"

3. **Экспорт:**
   - Правый клик → "Export Current Diagram"
   - Выберите формат (PNG/SVG/PDF)

### Способ 3: Командная строка (для автоматизации)

1. **Установите Java:**
   - Скачайте с https://www.java.com/

2. **Скачайте PlantUML JAR:**
   - https://plantuml.com/download
   - Или запустите `visualize_plantuml.bat` → вариант `3`

3. **Генерация PNG:**
   ```bash
   java -jar plantuml.jar diagrams\classes.puml
   java -jar plantuml.jar diagrams\sequence.puml
   java -jar plantuml.jar diagrams\architecture.puml
   ```

### Способ 4: IntelliJ IDEA / PyCharm

1. **Установите плагин:**
   - Settings → Plugins
   - Поиск "PlantUML Integration"
   - Установите и перезапустите

2. **Откройте .puml файл:**
   - Диаграмма отобразится автоматически
   - Можно экспортировать в PNG/SVG

### Способ 5: Docker (изолированно)

```bash
docker run -v %cd%:/data plantuml/plantuml diagrams/classes.puml
```

- ✅ Все атрибуты и методы классов

- ✅ Активация/деактивация участников
- ✅ Условные блоки (alt/else)
- ✅ Циклы (loop)
- ✅ Асинхронные вызовы
- ✅ Временная шкала
- ✅ Группировка по фазам

### Архитектура компонентов (architecture.puml):
- ✅ Пакеты и компоненты
- ✅ Базы данных
- ✅ Внешние системы
- ✅ Цветовое кодирование по типам
- ✅ Аннотации с описаниями

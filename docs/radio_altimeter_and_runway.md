# Радиовысотомер и проверка длины ВПП

## Радиовысотомер

Система теперь использует данные радиовысотомера (`RADIO_HEIGHT`) для более точного определения высоты над поверхностью.

### Преимущества:
- Точная высота над порогом ВПП
- Независимость от барометрического давления
- Критично для выравнивания и касания

### Использование в коде:
```python
radio_height = telemetry['position']['radio_height']  # футы
```

Радиовысотомер используется в фазах FINAL и LANDING для:
- Выпуска закрылков и шасси
- Определения момента перехода к посадке
- Выравнивания и касания

---

## Проверка длины ВПП

Система автоматически рассчитывает требуемую посадочную дистанцию и проверяет достаточность длины ВПП.

### Параметры конфигурации:
```python
ApproachConfig(
    ...
    runway_length=8000,           # длина ВПП (футы)
    runway_width=150,             # ширина ВПП (футы)
    runway_threshold_lat=55.48,   # широта порога
    runway_threshold_lon=37.52    # долгота порога
)
```

### Расчёт посадочной дистанции

Учитывается:
- Путевая скорость при касании
- Встречный/попутный ветер
- Состояние ВПП (сухая/мокрая/обледенелая)
- Вес самолёта

Формула:
```
Дистанция = (Скорость² / (2 × Замедление)) × Коэффициенты
```

### Коэффициент безопасности

По умолчанию используется запас 50% (коэффициент 1.5):
```
Требуемая дистанция с запасом = Базовая дистанция × 1.5
```

### Статусы проверки:

- **OK** - длина ВПП достаточна с запасом
- **WARNING** - длина ВПП достаточна, но запас минимальный
- **CRITICAL** - длина ВПП недостаточна!

### Пример лога:
```
FINAL: Distance to threshold 2.50nm, Radio height 1200ft, 
Crosswind: 8.5kt, Crab: 4.2°, VS: 650 fpm, 
Runway: OK (2500ft margin)
```

### Предупреждения:
При статусе CRITICAL система выдаёт предупреждение:
```
CRITICAL: Runway too short! Required: 6000ft, Available: 5000ft
```

## Расчёт расстояния до порога ВПП

Система рассчитывает точное расстояние до порога ВПП используя координаты:

```python
distance_to_threshold = navigation.calculate_distance_to_threshold(
    aircraft_lat, aircraft_lon, config
)
```

Это позволяет:
- Точно определить момент касания
- Рассчитать оставшуюся дистанцию
- Скорректировать глиссаду

---

## Рекомендации

1. **Всегда указывайте точные координаты порога ВПП**
2. **Проверяйте длину ВПП перед заходом**
3. **Учитывайте состояние полосы** (мокрая/обледенелая)
4. **Следите за предупреждениями CRITICAL**
5. **Оставляйте запас минимум 50%**

---

## Состояние ВПП

Коэффициенты увеличения дистанции:
- **Сухая (dry)**: 1.0× (базовая)
- **Мокрая (wet)**: 1.3× (+30%)
- **Обледенелая (icy)**: 2.0× (+100%)

Для изменения состояния ВПП в коде:
```python
required_distance = navigation.calculate_landing_distance(
    ground_speed=120,
    headwind=10,
    runway_condition='wet'  # 'dry', 'wet', 'icy'
)
```

# Автоматический контроллер тяги (Autothrottle)

## Обзор

Автоматический контроллер тяги (Autothrottle) обеспечивает точное управление двигателями во время фазы FINAL (глиссады) с учётом веса самолёта, конфигурации и ветра.

## Возможности

### 1. **PID контроллер скорости**
- Пропорциональная составляющая (P): быстрая реакция на отклонение
- Интегральная составляющая (I): устранение накопленной ошибки
- Дифференциальная составляющая (D): демпфирование колебаний

### 2. **Учёт веса самолёта**
- Автоматическая коррекция базовой тяги
- Тяжёлый самолёт → больше тяги
- Лёгкий самолёт → меньше тяги

### 3. **Учёт конфигурации**
- **Закрылки:** каждая позиция добавляет 15% тяги (компенсация сопротивления)
- **Шасси:** выпущенное шасси добавляет 10% тяги

### 4. **Коррекция на ветер**
- Встречный ветер → больше тяги
- Попутный ветер → меньше тяги
- ~1% тяги на каждые 5 узлов ветра

### 5. **Интеграция с vJoy**
- Прямое управление тягой через виртуальный джойстик
- Более точный контроль, чем через SimConnect
- Автоматическое переключение между vJoy и SimConnect

## Как это работает

### Алгоритм расчёта тяги

```
1. Базовая тяга = 50% + коррекция_на_вес + коррекция_на_конфигурацию
2. Коррекция_на_ветер = встречный_ветер × 0.002
3. PID_коррекция = Kp×ошибка + Ki×интеграл + Kd×производная
4. Итоговая_тяга = Базовая + Ветер + PID
5. Ограничение скорости изменения (плавность)
6. Применение через vJoy или SimConnect
```

### Пример расчёта

**Условия:**
- Вес: 7000 фунтов (тяжелее референсного 5000)
- Закрылки: позиция 3 (посадочная конфигурация)
- Шасси: выпущено
- Встречный ветер: 15 узлов
- Текущая скорость: 115 узлов
- Целевая скорость: 120 узлов

**Расчёт:**
```
Базовая тяга:
  50% + (7000-5000)×0.00002 = 50% + 4% = 54%
  
Конфигурация:
  Закрылки: 3 × 15% = 45%
  Шасси: 10%
  Итого: 54% + 45% + 10% = 109% → ограничено до 100%
  
Коррекция на ветер:
  15 узлов × 0.002 = 3%
  
PID коррекция:
  Ошибка: 120 - 115 = 5 узлов
  P: 0.05 × 5 = 0.25 (25%)
  I: 0.01 × интеграл
  D: 0.02 × производная
  
Итоговая тяга: ~75-80% (после ограничений)
```

## Конфигурация

### Параметры по умолчанию

```python
AutothrottleConfig(
    # PID коэффициенты
    kp=0.05,  # Пропорциональный
    ki=0.01,  # Интегральный
    kd=0.02,  # Дифференциальный
    
    # Ограничения
    max_throttle=1.0,
    min_throttle=0.0,
    max_throttle_rate=0.1,  # Максимальная скорость изменения
    
    # Допуски
    target_speed_tolerance=5.0,  # ±5 узлов = стабильно
    
    # Коррекция на конфигурацию
    flaps_drag_factor=0.15,  # 15% на позицию закрылков
    gear_drag_factor=0.10,   # 10% при выпущенном шасси
    
    # Коррекция на вес
    weight_reference=5000.0,  # Референсный вес (фунты)
    weight_factor=0.00002     # Коэффициент влияния веса
)
```

**Лёгкие самолёты (Cessna 172):**

## Использование

### Автоматическая активация

Autothrottle автоматически активируется при переходе к фазе FINAL:

```python
# В main.py
if distance < 8 and abs(altitude - required_alt) < 300:
    self.phase = ApproachPhase.FINAL
    self.autothrottle.activate(initial_throttle=0.5)
```

### Ручное управление

```python
from modules.autothrottle import AutothrottleController

# Создание контроллера
autothrottle = AutothrottleController()

# Активация
autothrottle.activate(initial_throttle=0.5)

# Расчёт тяги
throttle_data = autothrottle.calculate_throttle(
    telemetry=telemetry,
    target_speed=120,  # узлы
    wind_data=wind_data,
    aircraft_weight=5000  # фунты
)

# Применение
control.set_throttle(throttle_data['throttle'])

# Деактивация
autothrottle.deactivate()
```

## Интеграция с vJoy

### Автоматическое переключение

Система автоматически использует vJoy если доступен:

```python
# При подключении
if self.virtual_joystick.connect():
    self.vjoy_throttle = VJoyThrottleIntegration(self.virtual_joystick)
    self.vjoy_throttle.enable()

# При применении тяги
if self.vjoy_throttle and self.vjoy_throttle.enabled:
    self.vjoy_throttle.set_throttle(throttle)  # Через vJoy
else:
    self.control.set_throttle(throttle)  # Через SimConnect
```

### Преимущества vJoy

- **Точнее:** прямое управление осью тяги
- **Быстрее:** меньше задержка
- **Плавнее:** нет дискретных шагов SimConnect

## Фазы работы

### 1. INITIAL / INTERMEDIATE
Autothrottle **неактивен**, используется базовое управление тягой.

### 2. FINAL (глиссада)
Autothrottle **активен**:
- Поддерживает целевую скорость захода
- Компенсирует ветер
- Учитывает вес и конфигурацию
- Плавно корректирует тягу

### 3. LANDING (выравнивание)
Autothrottle **деактивирован**, управление передаётся FlareController:
- 50 ft: начало снижения тяги
- 30 ft: постепенное снижение
- 10 ft: ~10% тяги
- 5 ft: idle (5%)

### 4. GO AROUND
Autothrottle **деактивирован**, полный газ:
- Через vJoy или SimConnect: 100% тяги
- Немедленное применение

## Устранение проблем

### Тяга слишком агрессивная

**Проблема:** Резкие изменения тяги, колебания скорости

**Решение:**
```python
config.kp = 0.03  # Уменьшить пропорциональный коэффициент
config.max_throttle_rate = 0.05  # Ограничить скорость изменения
```

### Тяга слишком медленная

**Проблема:** Долго реагирует на изменения

**Решение:**
```python
config.kp = 0.08  # Увеличить пропорциональный коэффициент
config.max_throttle_rate = 0.15  # Разрешить быстрее изменение
```

### Накопление ошибки

**Проблема:** Постоянное отклонение скорости в одну сторону

**Решение:**
```python
config.ki = 0.02  # Увеличить интегральный коэффициент
```

### Колебания скорости

**Проблема:** Скорость постоянно колеблется вокруг целевой

**Решение:**
```python
config.kd = 0.04  # Увеличить дифференциальный коэффициент (демпфирование)
```

# Инструкции по установке библиотек для звуковых предупреждений

## Требуемые библиотеки

Для работы системы звуковых предупреждений необходимо установить:

```bash
pip install gtts pygame
```

### gTTS (Google Text-to-Speech)
- Генерация голосовых сообщений из текста
- Использует Google TTS API
- Создает MP3 файлы с голосовыми предупреждениями

### pygame
- Воспроизведение аудио файлов
- Управление громкостью
- Проверка статуса воспроизведения

## Установка

```bash
# Установка обеих библиотек
pip install gtts pygame

# Или через requirements.txt
pip install -r requirements.txt
```

## Проверка установки

```python
python -c "from gtts import gTTS; import pygame; print('Audio libraries OK')"
```

## Если библиотеки не установлены

Система автоматически определит отсутствие библиотек и продолжит работу без звуковых предупреждений.
В логах появится сообщение:
```
WARNING - Audio system not available. Install: pip install gtts pygame
```

## Предварительная генерация аудио

При первом запуске система автоматически сгенерирует все звуковые файлы в папку `audio_alerts/`:
- WINDSHEAR_CRITICAL.mp3
- WINDSHEAR_WARNING.mp3
- SINK_RATE.mp3
- TERRAIN.mp3
- STALL.mp3
- MINIMUMS.mp3
- RETARD.mp3
- TOO_LOW_GEAR.mp3
- TOO_LOW_FLAPS.mp3

Это займет несколько секунд при первом подключении к MSFS.

## Управление звуковыми предупреждениями

В GUI есть чекбокс "Audio Alerts" для включения/выключения звука во время работы.

## Примечания

- Требуется интернет-соединение для первой генерации аудио (gTTS использует Google API)
- После генерации файлы сохраняются локально и интернет не требуется
- Громкость по умолчанию: 100%
- Между повторными предупреждениями есть cooldown 5 секунд

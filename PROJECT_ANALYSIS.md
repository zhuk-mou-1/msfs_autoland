## ⚠️ ИЗВЕСТНЫЕ ОГРАНИЧЕНИЯ

1. **MobiFlight WASM** - требуется для полной функциональности кастомных самолётов
   - Fallback на SimConnect работает, но с ограничениями

2. **База данных аэропортов** - содержит только 8 аэропортов
   - Легко расширяется добавлением в JSON

3. **vJoy** - опционален, но рекомендуется для точного управления
   - Система работает и без vJoy

4. **Ручная настройка** - требует выбора параметров захода
   - Auto-detect из MSFS упрощает процесс
